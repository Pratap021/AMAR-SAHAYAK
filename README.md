# AMAR SAHAYAK

AMAR SAHAYAK is an AI-powered multilingual assistant chatbot built using Python and RAG (Retrieval-Augmented Generation) architecture.

## Features

- AI Chatbot System
- RAG-based Intelligent Response Generation
- WhatsApp Bot Integration
- Translation Support
- Session Handling
- Embedding-based Semantic Search
- CSV Dataset Processing
- Tokenizer API Integration

## Technologies Used

- Python
- NLP
- RAG Architecture
- Sentence Transformers
- Pandas
- Flask/FastAPI
- WhatsApp API
- Embeddings
- Vector Search
